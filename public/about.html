About the Disrupt Sports Product Database...

This is a simple in-house, page for employees of Disrupt Sports to find and browse the product catalogue....

Logging on

As this information is for Disrupt Sports employees only, you have to log-on, and an administrator has to grant you access to read.

  • First log-on with your Google account.
  • Request that you be made a 'reader' by an 'admin' user.
  • Refresh the page.

Searching

This is the main purpose for this site, you can quickly find the details about the product range here.

  • Searching operates through the name or quality specified for each item in the database. The result will be shown as a list of categories that contain all the found items.
  • If you want to search for a 'quality' type or item specific things, don't forget to select to search 'items'.
  • Searching does not use boolean operators such as 'and', 'or', 'not'
  • Searching is not case-sensitive

Enhancements

As a work-in-progress, there are a number of additions that could be made.

  • Add a 'catalogue' page showing thumbnails of all available products.
  • Add an admin function to delete items in the database that are not in the imported spreadsheet.
  • Add an admin function to download the database to a new spreadsheet to modify and re-import.
  • Add the ability to download the data to an offline document (word or pdf for printing) of available items.
  • Brackets might cause a problem with searching - need to remove them from search terms really.
  • Names and quality are currently readonly (because they are referenced by other items in the DB) can make these editable.
  • Multiple images per item / category to show a slide-show might be nice.
  • An iPad app to browse the data.
  • An Android app to browse the data.
  • Prices are uneditable, would be nice to allow edit of these, and maybe adding and deleting quantities
  • You can enter a URL to an image, but would be nice to allow the upload of images to the cloud store
  • Add a customer user role to allow the browsing of the database (without seeing prices?)
  • Items can only be added currently through importing a new data spreadsheet, could add 'addition' options
  • Items can only be deleted currently through clearing and starting again, could add 'delete' options
